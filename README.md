# Charity Donation Tracker

A transparent, blockchain-powered command-line application where every donation to a charity campaign is recorded on-chain and can never be hidden or changed.

Built with **Solidity** smart contracts on a local **Ganache** Ethereum network, interacted with via **Python** and **web3.py**.

## Overview

The Charity Donation Tracker is a decentralised application (DApp) that separates users into two roles:

- **Admin** — Can add/update charity campaigns, mint Donor Coins (DNRC), pause/resume the contract, and transfer ownership.
- **Normal User** — Can register a display name, donate ETH to campaigns, view activity history, and check balances.

All data lives on the blockchain, making donations transparent and tamper-proof.

## Setup Instructions

### Step 1: Clone the Repository

```bash
git clone <repository-url>
cd charity-donation-tracker
```

### Step 2: Install Python Dependencies

```bash
pip install -r requirements.txt
```

### Step 3: Start Ganache

Open **Ganache GUI** or run **Ganache CLI**:

```bash
ganache-cli -p 7545
```

Make sure it is running on `http://127.0.0.1:7545` with at least 5 accounts.

### Step 4: Deploy the Contracts

```bash
python scripts/deploy.py
```

This will:
- Compile `CharityCampaigns.sol` and `DonorCoin.sol`
- Save ABIs to the `abis/` folder
- Deploy both contracts to Ganache
- Seed 3 sample campaigns
- Mint test Donor Coins
- Save contract addresses to `config.json`

### Step 5: Run the Terminal App

```bash
python scripts/terminal_app.py
```

## Admin Guide

### Accessing the Admin Menu

From the main menu, select the **Admin menu** option. You will be prompted for the admin password/key.

### Admin Actions

| Action | Description |
|--------|-------------|
| **Add Campaign** | Create a new charity campaign with a name, description, and fundraising goal |
| **Batch Add Campaigns** | Add multiple campaigns in a single transaction |
| **Mint Donor Coins** | Mint DNRC tokens to any address as a reward for donations |
| **Pause Contract** | Emergency stop — blocks all user donations and registrations |
| **Resume Contract** | Lift the pause and allow normal operations again |
| **Transfer Ownership** | Hand over admin rights to another wallet address |

### Important Notes

- Only the wallet that deployed the contract is the initial admin.
- After transferring ownership, the old admin loses all privileges immediately.
- Pausing the contract does **not** affect view functions (reading data), only state-changing actions.

## User Guide

### Registering

When you first launch the terminal app and select your wallet, you will be prompted to register a display name. This name is stored on-chain and shown in the menu header.

- Registration is **one-time only** — you cannot change your name after registering.

### Donating to a Campaign

1. Select **"Donate to a campaign"** from the menu.
2. View the list of available campaigns.
3. Enter the campaign ID and the amount of ETH to donate.
4. The donation is recorded on the blockchain permanently.

### Checking Balances

Select **"Check balance"** to see:
- Your **Donor Coin (DNRC)** balance
- Your **ETH** balance

Both are displayed side-by-side in a clean, human-readable format.

### Viewing Activity History

Select **"View my activity history"** to see a table of every action your address has performed:
- Block number
- Action type (Donated, Received Coins, Registered, etc.)
- Value in ETH
